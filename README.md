oneKit
======

一系列基于jQuery和require.js的UI组件。

组件依赖于：jQuery, require.js, ionicons.css

## 目录结构：
        ├───assets             // 依赖库
        ├───build
        ├───dist               // 真正的组件
        │   ├───css
        │   ├───fonts
        │   └───js
        ├───docs               // 说明文档
        ├───examples           // 演示例子
        │   └───js
        │       └───templates
        └───src                // 源文件
            ├───fonts
            ├───js
            │   ├───nls
            │   └───_base
            └───scss
