oneKit
======

一系列基于jQuery和require.js的UI组件。

组件依赖于：jQuery, require.js, ionicons.css

## Code Example
目录结构：

├───assets
├───build
├───dist
│   ├───css
│   ├───fonts
│   └───js
├───docs
├───examples
│   └───js
│       └───templates
└───src
    ├───fonts
    ├───js
    │   ├───nls
    │   └───_base
    └───scss
